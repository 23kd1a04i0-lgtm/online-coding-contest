# 🚀 Online Coding Contest Platform

## 📌 Overview

The **Online Coding Contest Platform** is a web-based application developed using **Python, Flask, SQLite, HTML, CSS, and Bootstrap**. It allows users to register, log in, participate in coding contests, view leaderboards, and manage their profiles.

## ✨ Features

* 👤 User Registration & Login
* 🏠 User Dashboard
* 💻 Coding Contest Page
* 🏆 Leaderboard
* 📜 Contest History
* 👨‍💼 User Profile
* 🔒 Admin Panel
* 📱 Responsive UI using Bootstrap

## 🛠️ Technologies Used

* Python
* Flask
* SQLite
* HTML5
* CSS3
* Bootstrap 5
* Gunicorn
* Render
